## What is Angular?

Angular is a comprehensive, open-source framework for building dynamic web applications using TypeScript. It provides everything you need: routing, forms, HTTP client, testing tools, and more.

**Key Characteristics:**
- Full-featured framework (not just a library)
- Strong typing with TypeScript
- Component-based architecture
- Built-in dependency injection
- Reactive programming with RxJS

## Quick Start (5 minutes)

### 1. Install Node.js
Visit [nodejs.org](https://nodejs.org) and install the LTS version.

### 2. Install Angular CLI
```bash
npm install -g @angular/cli@latest
```

### 3. Create Your First Project
```bash
ng new my-app
cd my-app
ng serve --open
```

Your application will open at `http://localhost:4200/`
